# Development Assessment for Gamebasics

This a submission for a job interview at gamebasics B.V. using vite.

how to run:
- pages: view the deploment at https://jasperfirecai2.github.io/gamebasics-assessment/
- static: download the build archive and unpack it in your choice of webserver
- live preview: download the code, install required dependencies with `npm install`, run `npm run preview`
- dev server: download the code, install all dependencies with `npm install`, run  `npm run dev`
